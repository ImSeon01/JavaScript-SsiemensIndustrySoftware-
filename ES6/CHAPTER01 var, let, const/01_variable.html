* 변수

- 데이터를 저장하는 공간

자바스크립트 변수를 선언하는 4가지 방법

  1. var
  2. let
  3. const
  4. nothing

1. var

- "var"는 1995~2015년 자바스크립트 코드에서 사용한 변수를 선언하는 키워드
- 하위호환성을 위해서는 var 변수를 사용해야 함
- 많은 범위에서 공유되는 최상위 변수에 사용
- "var"는 함수의 코드 블록만을 스코프로 인정하기 때문에, 함수 내부 이외에 생성한 변수는 모두 전역 변수

2. let

- let(및 const) 키워드로 선언된 변수는 블록 범위
- 즉, 선언된 블록과 해당 하위 블록 내에서만 사용
